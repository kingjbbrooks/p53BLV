# p53BLV

> Whole-Genome Binding Locations Viewer of Human p53

## Install

```bash
$ npm i
```
or
```bash
$ yarn install
```

## Config

```bash
$ cp option.sample.json option.json

# fill in all information in the configuration file
$ vi option.json
```

## Start

```bash
$ npm start
```
or
```bash
$ yarn start
```
